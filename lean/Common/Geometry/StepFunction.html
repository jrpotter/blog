    Type
    • p : Set.Partition ℝ
    • toFun : Fin (List.NonEmpty.length p.ivls) → ℝ

    A function f, whose domain is a closed interval [a, b], is a StepFunction if there exists a Partition P = {x₀, x₁, …, xₙ} of [a, b] such that f is constant on each open subinterval of P.

    Instead of maintaining a function from [a, b] to ℝ, we instead maintain a function that maps each Partition index to some constant value.
