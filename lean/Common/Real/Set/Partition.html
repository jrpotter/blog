    Common.Real.Set.Partition #

    A description of a partition as defined in the context of stepwise functions. Refer to [^1] for more information.

    [^1]: Apostol, Tom M. Calculus, Vol. 1: One-Variable Calculus, with an Introduction to Linear Algebra. 2nd ed. Vol. 1. 2 vols. Wiley, 1991.

    Type
    • xs : List ℝ
    • sorted : List.Sorted LT.lt xs
    • has_min_length : List.length xs ≥ 2

    A Partition is some finite subset of [a, b] containing points a and b.

    It is assumed that the points of the Partition are distinct and sorted. The use of a List ensures finite-ness.
