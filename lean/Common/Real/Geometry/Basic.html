    def Real.similar (S : Set (ℝ × ℝ)) (T : Set (ℝ × ℝ)) :
    Prop

    Two sets S and T are similar iff there exists a one-to-one correspondence between S and T such that the distance between any two points P, Q ∈ S and corresponding points P', Q' ∈ T differ by some constant α. In other words, α|PQ| = |P'Q'|.

    Equations
    • Real.similar S T = ∃ f, Function.Bijective f ∧ ∃ s, ∀ (x y : ℝ × ℝ), x ∈ S ∧ y ∈ T → s * Real.dist x y = Real.dist (f x) (f y)

    def Real.congruent (S : Set (ℝ × ℝ)) (T : Set (ℝ × ℝ)) :
    Prop

    Two sets are congruent if they are similar with a scaling factor of 1.

    Equations
    • Real.congruent S T = ∃ f, Function.Bijective f ∧ ∀ (x y : ℝ × ℝ), x ∈ S ∧ y ∈ T → Real.dist x y = Real.dist (f x) (f y)
