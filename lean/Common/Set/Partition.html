    structure Set.Partition (α : Type u_1) [inst : LT α] :
    Type u_1
    • ivls : List.NonEmpty (Set.Interval α)
    • connected : ∀ (I : Prop), (I ∈ List.pairwise (List.NonEmpty.toList ivls) fun x x_1 => x.right = x_1.left) → I

    A Partition is a finite subset of [a, b] containing points a and b.

    We use a List.NonEmpty internally to ensure the existence of at least one Interval, which cannot be empty. Thus our Partition can never be empty. The intervals are sorted via the connected proposition.
