    A representation of a range of values.

    source
    ink
    structure Set.Interval (α : Type u_1) [inst : LT α] :
    Type u_1
    • left : α
    • right : α
    • h : left < right

    An interval defines a range of values, characterized by a "left" value and a "right" value. We require these values to be distinct; we do not support the notion of an empty interval.
