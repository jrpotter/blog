      instance Set.Intervals.Partition.instMembershipPartition {α : Type u_1} [inst : Preorder α] [inst : DecidableRel LT.lt] :
      Membership α (Set.Intervals.Partition α)

      An object x is a member of a Partition p if x is an endpoint of p or a subdivision point of p.

      Notice that being a member of p is different from being a member of some (sub)interval determined by p.

      Equations
