    • A function whose domain is a closed interval [a, b].

      function : (x : α) → x ∈ Set.Icc partition.a partition.b → α

    • Ensure the function is constant on each open subinterval of p.

      const_open_subintervals : ∀ {I : Set α} {y : α} (hI : I ∈ Set.Intervals.Partition.openSubintervals partition), ∃ c, ∀ (hy : y ∈ I), function y (_ : y ∈ Set.Icc partition.a partition.b) = c

    A function f, whose domain is a closed interval [a, b], is a StepFunction if there exists a Partition P = {x₀, x₁, …, xₙ} of [a, b] such that f is constant on each open subinterval of P.
