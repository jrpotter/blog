    axiom Real.Geometry.Area.𝓜 :
    Set (Set (ℝ × ℝ))

    All measurable sets, i.e. sets in the plane to which an area can be assigned.

    The existence of such a class is assumed in the axiomatic definition of area introduced by Apostol. He denotes this set of sets as 𝓜.

    def Real.Geometry.Area.forall_subset_between_step_imp_le_between_area (k : ℝ) (Q : Set (ℝ × ℝ)) :
    Prop

    Exhaustion property.

    Let Q be a set that can be enclosed between two step regions S and T, so that (1.1) S ⊆ Q ⊆ T. If there is one and only one number k which satisfies the inequalities a(S) ≤ k ≤ a(T) for all step regions S and T satisfying (1.1), then Q is measurable and a(Q) = k.
